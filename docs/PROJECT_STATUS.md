## 已知限制

1. **记忆系统**
   - 情景记忆使用文件存储，大量记忆时性能可能下降
   - 语义记忆的关键词匹配较简单，复杂查询效果有限

2. **自进化系统**
   - 技能生成依赖 LLM 质量，可能生成不可用的技能
   - 沙箱验证较简单，复杂技能可能验证不准确

3. **CLI**
   - 无彩色输出
   - 无进度指示
   - 命令较少

4. **MiniMax 集成**
   - 流式响应存在重复输出问题（已临时禁用流式输出）
   - 需要使用 Bearer 认证而非标准 x-api-key

5. **测试**
   - 缺少单元测试
   - 缺少性能基准测试
   - 缺少真实 API 集成测试
